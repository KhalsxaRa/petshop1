# petshop

## Deskripsi

Proyek ini adalah website sederhana untuk toko hewan peliharaan bernama **A Pet Friend**. Website ini dibuat menggunakan HTML, CSS (Bootstrap & custom), dan JavaScript. Fitur utama meliputi:
- Navigasi responsif
- Halaman beranda dengan deskripsi toko
- Bagian produk dengan fitur tambah/kurang jumlah dan keranjang belanja
- Bagian tentang kami
- Blog dengan artikel seputar hewan peliharaan
- Kontak dan footer dengan media sosial

## Struktur File

- `src/home-ans.html` — Halaman utama website
- `src/home.css` — File CSS untuk styling tambahan
- `src/home-1.js` — Script JavaScript untuk interaksi produk & keranjang
- Gambar dan aset lain berada di folder `src/`


## Fitur Utama

- **Navigasi**: Menu responsif dengan Bootstrap.
- **Produk**: Daftar produk dengan tombol tambah/kurang dan keranjang.
- **Tentang Kami**: Penjelasan visi dan misi toko.
- **Blog**: Artikel informatif seputar hewan peliharaan.
- **Kontak**: Informasi alamat, telepon, email, dan media sosial.
